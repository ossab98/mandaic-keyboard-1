# Mandaic Keyboard

- A keyboard for typing in the
  [Mandaic Language](https://en.wikipedia.org/wiki/Mandaic_script), which is a
  descendent of Aramaic.

- Mandaic is related to Semitic languages such as Assyrian and Syriac.
